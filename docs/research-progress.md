## Research Challenges

1. **Data Variations:** Different sources report different numbers for the same operator
2. **Year Differences:** Data ranges from 2020-2025, making comparisons difficult
3. **Metric Differences:** Some report "subscribers", others "TV households", "customers", or "homes"
4. **Reporting Methods:** Some operators report total customers vs. TV-only customers
