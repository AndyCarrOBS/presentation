## Challenges Encountered

1. **Data Availability:** Many operators don't publicly disclose subscriber numbers
2. **Data Consistency:** Different sources report different numbers for the same operator
3. **Date Variations:** Data from different years makes comparison difficult
4. **Mergers & Acquisitions:** Many operators have merged (e.g., Unitymedia â†’ Vodafone, Ziggo â†’ UPC)
5. **Definition Variations:** "Subscribers" vs "TV Homes" vs "Customers" - definitions vary
6. **Standards vs Operators:** Some entries are technical standards (DVB-T2, CableHDReady) not operators
